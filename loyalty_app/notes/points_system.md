# Points

### Points can be input from any vendor csv
Assign a point value for the program and use a multiplier to add points to an account, using data from any vendor.

**Sample Vendor Data**
| email            | moneySpent |
| ---------------- | ----------- |
| j@reverbsoul.com | $50         |
| doobie@gmail.com | $200        |
| hashish@mail.com | $80         |
| jon@cosmic.com   | $10         |

multiplier: .5

| email            | points |
| ---------------- | ------ |
| j@reverbsoul.com | 25     |
| doobie@gmail.com | 100    |
| hashish@mail.com | 40     |
| jon@cosmic.com   | 5      |
