# JS Types

Types are values assigned to a variable as the value "1" is in let money = 1, or a value passed to a fn such as console.log('this is a String Type')

There are 6 types in JS:

- String
  - Text
- Number
  - Numerical Value
- Boolean
  - True or False
- Null
- Undefined
- Symbol
  - Unique ID
- Object
